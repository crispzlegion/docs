## Architecture

Solana integration uses Fairblock's cross-chain messaging layer to connect Solana programs with FairyRing's confidential computing primitives.

## Planned Features

| Feature | Description |
| --- | --- |
| **Encrypted instructions** | Submit encrypted Solana instructions for future execution |
| **Confidential transfers** | Encrypted SPL token transfers |
| **Sealed-bid auctions** | Confidential bidding on Solana |
| **MEV protection** | Transaction encryption for front-running prevention |
