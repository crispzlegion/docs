## What FairyKit Provides

| Feature | Description |
| --- | --- |
| **Encryption helpers** | Client-side encryption using FairyRing's Master Public Key |
| **Contract interfaces** | Onchain contracts for submitting and receiving encrypted data |
| **Key management** | Automatic MPK fetching and rotation |
| **Callback system** | Automatic execution of decrypted transactions |
| **Cross-chain relay** | IBC-based communication with FairyRing |
