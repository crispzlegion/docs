## Integration Methods

Each ecosystem uses the appropriate integration path:

- **EVM chains**: Precompiled contracts + FairyKit for EVM
- **Cosmos chains**: Native IBC + FairyKit modules
- **Solana**: Cross-chain messaging layer
- **FairyRing native**: Direct access to all confidential computing primitives
