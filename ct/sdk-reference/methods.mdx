## Client Initialization

```typescript
import { ConfidentialTransferClient } from "@fairblock/stabletrust";

const client = new ConfidentialTransferClient({
  network: "ethereum-sepolia",
  signer: walletSigner,
});

await client.setupAccount();
```

## Deposit

Convert public ERC20 tokens into encrypted confidential balances.

```typescript
const tx = await client.deposit({
  token: "USDC",
  amount: "1000",
});

console.log("Deposit tx:", tx.hash);
```

### Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `token` | string | Token symbol (USDC, USDT, USDG) |
| `amount` | string | Amount in token units |

## Transfer

Execute private transfers that hide amounts and balances.

```typescript
const tx = await client.transfer({
  to: "0xRecipient",
  amount: "250",
});
```

### Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `to` | string | Recipient address or numeric account ID |
| `amount` | string | Transfer amount |

## Withdraw

Move funds from the confidential layer back to public ERC20 tokens.

```typescript
const tx = await client.withdraw({
  token: "USDC",
  amount: "500",
  destinationChain: "ethereum-sepolia",
});
```

### Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `token` | string | Token symbol |
| `amount` | string | Withdrawal amount |
| `destinationChain` | string | Target chain identifier |

## Get Balance

Query encrypted balance (decrypted locally).

```typescript
const balance = await client.getBalance("USDC");
console.log("Balance:", balance.decrypted);
```

## Get Deposit Status

```typescript
const status = await client.getDepositStatus(tx.hash);
// Returns: "pending" | "confirmed" | "credited"
```
