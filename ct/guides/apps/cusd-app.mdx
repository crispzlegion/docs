CUSD is a confidential USD stablecoin built on Fairblock's confidential transfer infrastructure.

## Overview

CUSD enables encrypted stablecoin transactions where amounts and balances remain hidden onchain while addresses stay transparent for compliance.

## Features

- Encrypted balances using Twisted ElGamal homomorphic encryption
- Cross-chain deposits and withdrawals
- Selective disclosure for regulatory compliance
- Real-time mobile transaction speeds

## Getting Started

<Steps>
  <Step title="Access the App">
    Navigate to the CUSD application interface.
  </Step>
  <Step title="Connect Wallet">
    Connect your EVM-compatible wallet.
  </Step>
  <Step title="Deposit">
    Deposit supported stablecoins to create encrypted balances.
  </Step>
  <Step title="Transact">
    Send and receive CUSD with fully encrypted amounts.
  </Step>
</Steps>
