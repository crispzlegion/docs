## Connection Details

| Protocol | Endpoint (Testnet) |
| --- | --- |
| **RPC** | `https://rpc.testnet.fairblock.network:443` |
| **REST** | `https://api.testnet.fairblock.network` |
| **gRPC** | `grpc.testnet.fairblock.network:9090` |
| **WebSocket** | `wss://rpc.testnet.fairblock.network/websocket` |

## Authentication

Testnet endpoints are publicly accessible. No API keys required.

| Tier | Requests/min |
