
## Quick Example

Send a confidential stablecoin transfer in just a few lines:

```typescript
import { ConfidentialTransferClient } from "@fairblock/stabletrust";

// Initialize the client
const client = new ConfidentialTransferClient({
  network: "ethereum-sepolia",
  signer: walletSigner,
});

// Set up your confidential account (one-time)
await client.setupAccount();

// Deposit tokens into the confidential layer
await client.deposit({
  token: "USDC",
  amount: "1000",
});

// Send a confidential transfer — amount and balance are encrypted
await client.transfer({
  to: "0xRecipientAddress",
  amount: "250",
});

// Check your encrypted balance
const balance = await client.getBalance("USDC");
console.log("Decrypted balance:", balance.decrypted);
```
