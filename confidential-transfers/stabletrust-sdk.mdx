## Core Operations

| Operation | Description |
| --- | --- |
| **Deposit** | Convert public ERC20 tokens into encrypted confidential balances |
| **Transfer** | Execute private transfers that hide amounts and balances |
| **Withdraw** | Move funds from the confidential layer back to public ERC20 tokens |

## Quick Start

```typescript
import { ConfidentialTransferClient } from "@fairblock/stabletrust";

const client = new ConfidentialTransferClient({
  network: "ethereum-sepolia",
  signer: walletSigner,
});

await client.setupAccount();

await client.deposit({ token: "USDC", amount: "1000" });

await client.transfer({ to: "0xRecipient", amount: "250" });

const balance = await client.getBalance("USDC");
console.log("Balance:", balance.decrypted);
```
